## 🆘 Troubleshooting

### "I don't have git installed"
```
Download: https://git-scm.com/download/win
Run installer, restart PowerShell
```

### "Python not found"
```
Make sure python is in PATH:
python --version
If not, reinstall Python and check "Add to PATH"
```

### "I don't know my GitHub URL"
```
The deploy script will ask you for it
Create one at: https://github.com/new
Use HTTPS URL: https://github.com/YOUR_USERNAME/jimmy-ai-bot.git
```

### "Deployment fails on Render"
```
1. Check Render logs in dashboard
2. Verify GOOGLE_API_KEY is set (real key, not "test_google_key")
3. Check SECRET_KEY is long enough (32+ characters)
4. Verify PUBLIC_BASE_URL matches your Render service name
```
