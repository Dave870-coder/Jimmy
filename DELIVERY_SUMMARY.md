## 📊 What's Happening Behind the Scenes

When someone sends a message to your bot:

```
1. Message arrives on Telegram
2. Telegram sends webhook to your server
3. Jimmy processes the message
4. Routes to appropriate AI agent
5. Google AI generates response
6. Response sent back to Telegram user

⏱️ Total time: 1-5 seconds
```

**No polling. No delays. True real-time.** ✨
