## Project: Jimmy AI Bot Platform

An intelligent Telegram and WhatsApp bot platform powered by Google AI Studio, with a modern Next.js dashboard for real-time monitoring and management.

### Key Technologies
- **Backend**: FastAPI (Python)
- **Frontend**: Next.js 14 (TypeScript)
- **AI**: Google Generative AI (Gemini)
- **Database**: SQLite (local), PostgreSQL (production)
- **Hosting**: Render, Railway, or similar cloud platforms
- **Messaging**: Telegram Bot API, WhatsApp Cloud API

### Features
- Real-time Telegram webhook gateway
- Secure credential management via .env
- Live dashboard with analytics and bot status
- User and message tracking
- WhatsApp QR code authentication
- Google AI-powered responses
- Multi-platform support (Telegram, WhatsApp, Web)
