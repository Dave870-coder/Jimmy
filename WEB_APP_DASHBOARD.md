## ⚡ 3-Step Setup (3 minutes total)

### Step 1: Enable GitHub Pages (1 min)

```
GitHub → Repository Settings → Pages → 
Build and deployment: Select "GitHub Actions" → Save
```

✅ Done! GitHub Pages is enabled.

### Step 2: Push Code (1 min)

```bash
git push origin main
```

GitHub Actions automatically builds and deploys your dashboard.

### Step 3: Visit Your Dashboard (1 min)

Wait 2-3 minutes, then visit:

```
https://YOUR_USERNAME.github.io/jimmy-ai-bot/
```

✅ Your dashboard is LIVE! 🎉

## 🔧 Configuration (Optional)

### Connect API to Dashboard

For live metrics, set your API URL:

```
Settings → Secrets and variables → Actions
New repository secret:
  Name: NEXT_PUBLIC_API_BASE
  Value: https://your-bot-api.railway.app
```

Then push: `git push origin main`

Dashboard will now show live data from your bot! 📊

## 📞 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Dashboard not loading | Wait 5 min, hard refresh (Ctrl+Shift+R) |
| No data showing | Set NEXT_PUBLIC_API_BASE secret |
| Build failed | Check Actions tab for errors |
| Deployment taking long | First deploy can take 5-10 min |
